# LegalEase GovHack 2025 Presentation

## Overview
This repository contains a professional presentation for LegalEase - an AI-powered regulatory navigation platform created for GovHack 2025.

## Files Created
- `presentation.md` - Markdown version with full content and voiceover cues
- `presentation.html` - Interactive HTML slideshow using Reveal.js
- `PRESENTATION_README.md` - This documentation file

## How to View the Presentation

### Option 1: HTML Slideshow (Recommended)
1. Open `presentation.html` in a web browser
2. Use arrow keys or spacebar to navigate slides  
3. Press 'V' to toggle voiceover cues on/off
4. Auto-advance is set to 12 seconds per slide (2 minutes total)

### Option 2: Markdown Version
- Open `presentation.md` in any markdown viewer
- Contains all slide content plus voiceover scripts
